How it works: This page connects to your Vocal Bridge agent via LiveKit (WebRTC). The backend (server.py) calls POST /api/v1/token to get a LiveKit room token, then the browser connects directly for real-time audio. Live transcripts appear automatically via the built-in send_transcript data channel.
